# 📡 Stock-Bot: 你的美股 AI 全景雷达

这是什么？ 这是一个基于 Google Gemini 2.5 的智能金融 Agent。它不像传统工具那样只罗列数据，而是像一位宏观对冲基金经理一样，帮你阅读海量新闻，分析美债、美元、科技股之间的逻辑关联，并给出交易策略。

## ✨ 核心亮点
* **🧠 顶级思维模型**：内置桥水 (Bridgewater) 风格的 Prompt，能够识别市场定价偏差（Price-in）与真实信号。
* **⚖️ 智能权重修正**：自动修正经济数据权重（如：重服务业 PMI，轻制造业 PMI），防止被单一数据误导。
* **🔍 自动降噪**：从海量新闻中过滤噪音，只呈现实质性驱动资产定价的关键事件。
* **🦅 流动性双向追踪**：不仅看紧缩 (QT)，也敏锐捕捉扩表 (QE) 信号，全天候监控美联储资产负债表。
